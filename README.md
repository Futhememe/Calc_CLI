# Usage
<!-- usage -->
```sh-session
$ npm install -g calccli
$ calccli COMMAND
running command...
$ calccli (-v|--version|version)
calccli/0.0.0 win32-x64 node-v12.16.1
$ calccli --help [COMMAND]
USAGE
  $ calccli COMMAND
...
```
